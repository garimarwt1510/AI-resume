# Resuno — design tokens

Color:
- --ink: #0B0E1A (near-black navy background for dark sections)
- --indigo: #4F46E5 (primary)
- --violet: #8B5CF6 (secondary / gradient partner)
- --cyan: #22D3EE (AI accent — used sparingly for "scan" motifs, scores)
- --white: #FFFFFF
- --mist: #F6F7FB (light section background)
- --slate: #64748B (muted text)

Type:
- Display: "Sora" (geometric, techy, confident — headlines)
- Body: "Inter" (neutral, highly legible)
- Mono/data: "JetBrains Mono" (scores, percentages, badges — reinforces "AI reading data" feel)

Layout / signature element:
- Hero: split layout. Left = headline + CTA. Right = a floating "scan card" — a
  mock resume sheet with an animated cyan scan-line sweeping top to bottom,
  and glass score badges (Resume Score, ATS Score) orbiting/floating beside it.
  This is the signature motif, echoed smaller in the "How It Works" step icons
  and the analyzer page's loading state (scan-line reused = cohesion).
- Cards: glassmorphism (translucent white, blur, thin light border, soft shadow)
  over a subtle indigo/violet gradient mesh background.
- Radius: 20px cards, 999px pills for buttons/badges.
